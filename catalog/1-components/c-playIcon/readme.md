# Posizione
È possibile definire la posizione dell'icona attraverso le seguenti classi:

- `c-playIcon-center`
- `c-playIcon-bottom-right`

<br>
# Dimensioni
È possibile cambiare la dimensione dell'icona attraverso le seguenti classi:

- `c-playIcon-xxs`
- `c-playIcon-xs`
- `c-playIcon-sm`
- `c-playIcon-md`
- `c-playIcon-lg`
